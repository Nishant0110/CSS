What is the use of isNaN function?

⇨ The isNaN function in JavaScript is used to determine whether a value is NaN (Not-a-Number).

⇨ It returns a boolean value indicating whether the provided value is NaN.

Example:-

let value1 = 10;

let value2 = "Hi";

let value3 = NaN;

console.log(isNaN(value1)); // Output: false, as 10 is a number

console.log(isNaN(value2)); // Output: true, as "Hi" is not a number

console.log(isNaN(value3)); // Output: true, as NaN is not a valid number

What is Negative Infinity?

⇨ Negative Infinity is a special value that represents the mathematical concept of negative infinity.

⇨ It is often used to represent the result of a mathematical operation that exceeds the lower limit of representable values in JavaScript.

Example:-
