What are the advantages of Bootstrap?

  1. Responsive design
    • Bootstrap enables the creation of websites that are compatible with different devices and screen sizes.
  2. Speed of development
    • Bootstrap provides predefined design templates and classes that can save time and effort.
  3. Customizability
    • Bootstrap can be modified according to the needs and preferences of the project.
  4. Support
    • Bootstrap has a large and active community of developers and users who can provide help and feedback.

What is a Bootstrap Container and how does it work?

Bootstrap Containers are the most basic layout element in Bootstrap.

Bootstrap Containers are very essential and basic building blocks of bootstrap that wrap a page's content.

It's responsible for setting and aligning content within it according to viewport or given device.

containers are established the width for the layout to give the content.

A Container wraps a page's content, providing structure and alignment within a given device or viewport.

It ensures that content is contained, padded, and sometimes centered.

Basically, there are three types of container classes available in bootstrap:

  • Default-Container
  • Responsive-Container
  • Fluid-Container

    ⇨Family: The default font family is “Helvetica Neue”, Helvetica, Arial, sans-serif.

    ⇨Font Size: The default font size for the body element is 16px

    ⇨Font Weight: Headings have a default font weight of bold

    ⇨Text color: By default, text color is typically set to a dark color for better contrast against light backgrounds.

    ⇨Text decoration: By default, Bootstrap does not apply any text decoration such as underline or strikethrough.

    ⇨Line height: The default line height is set to 1.5, which provides adequate spacing between lines for improved readability.
