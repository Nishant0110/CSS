What is jQuery?

⇨JQuery is a fast, small, and feature-rich JavaScript library.

⇨ jQuery is a lightweight, "write less, do more", JavaScript library.

⇨ Query also simplifies a lot of the complicated things from JavaScript, like DOM manipulation.

⇨ many lines of code written in javascript, into a method that can be called with a single line of code.

How to Apply CSS Using JQuery, How to Add Class and Remove Class in Jquery , jQuery Animation?

  1. Applying CSS
    • To apply CSS styles using jQuery, you can use the css() method..

      • Syntex:

      $(selector).css({property:value})

      Example:

      $("#myElement").css("color", "blue");

  2. Adding and removing classes
    • To add and remove classes, you can use the addClass() and removeClass() methods.

      • Syntex:

      $('your-selector').addClass('your-class-name');

      $('your-selector').removeClass('your-class-name');

      Example:

      $("#myElement").addClass("highlight");

      $("#myElement").removeClass("highlight");

  3. jQuery Animation
    • The jQuery animate() method provides you a way to create custom animations.

      • Syntex:

      $(selector).animate({params}, speed, callback);

      Example:

      $( "#effect" ).animate({backgroundColor: "#aa0000",color: "#fff",width: 500},1000 );
