jQuery Selectors

jQuery Selectors are used to select HTML element(s) from an HTML document.

Consider an HTML document is given and you need to select all the <div> from this document. This is where jQuery Selectors will help.

jQuery Selectors can find HTML elements based on the following:

  • HTML element Name
  • Element ID
  • Element Class
  • Element attribute name
  • Element attribute value

    • Syntex:

    $(document).action(function(){ $(selector) });

    A jQuery selector starts with a dollar sign $ and then we put a selector inside the braces ().

    Here $() is called factory function, which makes use of following three building blocks while selecting elements in a given document:Element,id,class.

Event Methods

jQuery events are the actions that can be detected by your web application.

They are used to create dynamic web pages.

An event shows the exact moment when something happens.

These are some examples of events.

  • Mouse Events

    • The click() method attaches an event handler function to an HTML element.

    • The dblclick() method attaches an event handler function to an HTML element.

    • The mouseenter() method attaches an event handler function to an HTML element.

    • The mouseleave() method attaches an event handler function to an HTML element.

      The function is executed when the user clicks on the HTML element.

      Syntex:

      $("selector").click();
  • Keyboard Events

    • Keypress() event specifies that the key is pressed down.

    • Keyup() event specifies that the key is released.

    • Keydown() event handler to the “keydown” event, or trigger that event on an element.

    • Syntex:

      $(selector).keydown(function)
  • Form Events

    • The focus() method attaches an event handler function to an HTML form field.

    • The blur() method attaches an event handler function to an HTML form field.

    • The submit() event handler to the “submit” event, or trigger that event on an element.

    • Syntex:

      $(selector).blur(function)
  • Document/Window Events

    • load() Bind an event handler to the “load” event, or trigger that event on an element.

    • unload() method is used to unload a specific element. It attaches an event handler to unload event.

    • The scroll() method is used to trigger the scroll event or attach a function to run when scrolling occurs.

    • resize() method triggers a resize event when the size of the browser window changes. It binds the event handler to resize an event or trigger that event on an element.

    • Syntex:

      $(selector).unload(function)
